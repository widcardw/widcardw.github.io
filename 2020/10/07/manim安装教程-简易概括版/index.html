manim安装教程(简要概括版)

2020-10-07

本文档仅适用于旧版 cairo-backend 分支下的 manim，新版请参照官方文档 https://manim.wiki

其实你读到后面会发现这篇文章真的是太烂了。（写这么长根本不是简要概括啊喂）

无图警告：本篇博文没有任何一张图片，全部为操作的描述，如果遇到不会操作的，请根据给出的视频链接或百度搜索来寻找解决方案。

Linux 下安装

Installation

这里的安装过程其实是从这个视频里面看来的

首先先说明一下，Ubuntu18.04 版本用sudo apt-get install python3安装python3时，默认安装的是python3.6.9，可能会有一些问题，所以建议还是安装 python3.7 及以上的版本。

而 Ubuntu20.04 默认安装的是python3.8.2，这对于 manim 来说，应当是不会有太大问题的。

另外，在这里并不是特别推荐使用python3.9，由于版本比较新，有一些库还没有更新，所以在安装依赖库的时候可能会报各种错。这个大家就先看情况，能装就装，不能就算了。

Windows 下安装

如果你是 Windows 10 用户，那你还可以使用 WSL（Windows Subsystem for Linux）进行安装，那么安装步骤就跟上面的 Linux 下安装几乎一样了；如果你是 Windows 7 用户，那可就只能按照常规方法安装了呢。

如果你已经了解过 WSL，那也应该懂得怎样去安装以及配置一些环境等等。如果你还不会，推荐去看该视频已被删除。

装好 WSL 之后，回到上面，看 Linux 下安装去，或者也可以选择看这个 视频BV1Kv411i7zn。

Windows 下常规安装方法

  • 首先你的网速不能太差，包括 pip 源等项目。接着你可以去看看 视频BV14i4y1G7Yr 使用包管理工具scoop来进行安装。如果你按照这个视频安装成功了，那下面的基本上也就不用看了。

  • 如果你不想要使用包管理器来安装的话，那下面的方法就是按照 视频BV1W4411Z7Zt 进行安装。

  • 首先，安装 python。

    注意选择 3.7 及以上的版本，或者你可以使用 anaconda3。在安装过程中，一定一定一定要把“Add to Path”勾选上！！！如果你是 conda 用户的话那就按照需求照着百度进行一些初始化等操作吧。同时，检查一下 pip 是否安装成功。

  • 接着安装 cairo。

    到 https://pypi.org/project/pycairo/#files 去下载cairo，注意：python3.7 请下载cp37的版本，python3.8 请下载cp38的版本。你会得到一个.whl文件。
    然后，在你下载这个.whl文件的目录下打开终端（cmd 或 powershell 或 git-bash），使用下面的命令安装 pycairo。请注意自己的版本，不要盲目的直接复制下面的命令。

  • 安装 git，你可以直接在百度上搜索 Git for Windows，就可以按照要求安装。这里不详细展开。

  • 安装 ffmpeg

    你可以到官方下载页面，前提是你看得懂怎么下载。

选计科并且使用 WSL 安装的同学请注意了

WSL（即 Windows Subsystem for Linux）使用的是64位机，并且在未安装交叉编译时是无法运行32位程序的！计科的同学将来肯定会学习计算机系统/计算机组成原理，老师会要求你们使用 IA-32 Linux 指令集完成实验，但实验时需要编译出的可执行文件中32位的可执行文件会无法运行，也无法使用 gdb 进行单步调试！

据某个同学了解，微软官方也说 WSL 只支持 64 位的，那我们也没什么办法。我就是本来以为安装了 WSL 可以不装虚拟机 Ubuntu 的，结果最终还是装了虚拟机的那个。也就是说，你可能无法将你对计算机的一部分学习内容与 manim 放在同一屋檐下，还是有点可惜。
